# StPeters-App-MAUI
.NET MAUI has massively improved cross platform mobile development experience compared to Xamarin Forms. This is an updated version of a previous church application.

.NET MAUI app built for local Roman Catholic church. This is an all new .NET Maui version ported across from original 
xamarin forms app in separate St Peters repo. Targets Android/iOS using .NET MAUI (.NET 10). 
Working versions of this app are published in respective app stores:

https://play.google.com/store/apps/details?id=st_Peters_RC_Calgary.St_Peters_RC_Calgary - to be replaced <br/>
https://itunes.apple.com/us/app/st-peters-rc-calgary/id985813714

Its been in use in various versions since 2015, finally got around to porting across to newer .NET MAUI framework Nov 2025. It is licensed open source so feel free to clone and leverage to build a version for your own parish/diocese/app.

To deploy in own dev environment:
1. Clone Repo locally.
2. Open in Visual Studio 2026 with .NET Maui Add-in installed. 
3. Restore nuget packages
4. Build, run
